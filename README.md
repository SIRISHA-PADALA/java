# Online Voting System (Java)

## Description
This is a simple Java-based Online Voting System.
It allows users to vote and stores vote data using file handling.

## Technologies Used
- Java
- File Handling

## Files
- OnlineVotingSystem.java
- SetupImages.java
- votes_backup.txt

## How to Run
1. Compile:
   javac OnlineVotingSystem.java SetupImages.java
2. Run:
   java OnlineVotingSystem
